# CLAUDE.md

## 项目概述

本项目是一个 Claude Code SKILL，名为"say-it-simple"（说人话），用于将专业、晦涩的内容转换为普通人都能理解的简单语言。适用于学术论文、日常工作、技术分享、跨行业沟通等场景。

## SKILL 名称

- **内部名称**: 说人话
- **触发命令**: `/说人话`

## SKILL 文件位置

`say-it-simple.SKILL/say-it-simple.md`

## 核心功能

### 支持的场景

1. **学术论文** - 论文创新点、技术贡献的通俗化解释
2. **技术方案** - 向非技术人员解释技术架构和选型
3. **跨行业沟通** - 消除行业术语壁垒
4. **日常工作** - 邮件、文档、会议的通俗化表达
5. **演讲准备** - 专业内容的通俗化转化

### 输出格式

- **格式一**：标准人话版（通用场景）
- **格式二**：论文深入浅出版（学术场景）
- **格式三**：行业翻译版（跨行业沟通）

## 工作流程

当用户输入 `/说人话 <内容>` 时：
1. 读取 `say-it-simple.SKILL/say-it-simple.md` 获取技能定义
2. 分析内容类型和场景
3. 根据内容类型选择合适的输出格式
4. 生成通俗易懂的解释
5. 返回结构化的输出结果

## 输出要求

- 语言简洁明了
- 多用类比和具体例子
- 保持专业准确性
- 结构化呈现复杂内容
- 根据场景自适应风格
