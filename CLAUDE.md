# CLAUDE.md

## 项目概述

本项目是一个 Claude Code SKILL，名为"说人话"，用于将专业、晦涩的内容转换为普通人都能理解的简单语言。

## SKILL 名称

`说人话`

## 命令触发

使用 `/说人话 <内容>` 触发技能。

## 技能定义位置

`say-it-simple.SKILL/say-it-simple.md`

## 工作流程

当用户输入 `/说人话 <内容>` 时：
1. 读取 `say-it-simple.SKILL/say-it-simple.md` 获取技能定义
2. 按照技能要求，将用户提供的内容转换为通俗易懂的语言
3. 使用类比、例子等方式帮助理解
4. 输出转换后的内容

## 输出要求

- 语言简洁明了
- 多用类比和具体例子
- 假设受众是小学生或普通成年人
- 避免使用更多专业术语
