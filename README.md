# 说人话 SKILL

将任何高大上、专业的定义、描述、概念、演讲稿等，转换为普通人都能理解的表达。

## 项目背景

在日常生活和工作中，我们经常遇到各种专业术语、行业黑话、抽象概念等，这些内容往往让人摸不着头脑。本项目的目标是：**让每一个人都能理解任何复杂的概念**。

## 核心原则

1. **简单性**：用最简单、直白的语言解释复杂概念
2. **类比性**：用生活化的例子和类比来解释
3. **具体性**：避免抽象描述，用具体例子说明
4. **受众思维**：假设受众是小学生、初中生或完全不懂该领域的普通人

## SKILL 使用方法

```
/说人话 <需要转换的内容>
```

例如：
```
/说人话 区块链是一种去中心化的分布式账本技术
/说人话 请将以下内容用人话解释：...
```
