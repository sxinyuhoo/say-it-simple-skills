# 说人话 SKILL

将任何高大上、专业的定义、描述、概念、演讲稿、技术方案、论文创新点、行业术语等，转换为任何人都能理解的表达。

## 核心理念

> 让每一个人都能理解任何复杂的概念

无论你是：
- 研究人员需要向非专业人士解释论文创新点
- 技术人员需要向老板或客户解释技术方案
- 跨行业从业者需要理解其他领域的术语
- 普通人想搞懂那些"高大上"的概念

这个 SKILL 都能帮你做到"深入浅出"。

## 功能特点

- **多场景适配**：学术论文、日常沟通、技术分享、跨行业解释
- **类比 + 举例**：用生活化的例子帮助理解
- **准确性优先**：简单不等于错误
- **结构化输出**：复杂内容不再一团浆糊

## 使用方式

```
/说人话 <需要转换的内容>
```

例如：
```
/说人话 请用通俗语言解释这个论文创新点：本文提出一种基于注意力机制的改进Transformer模型...
/说人话 帮我用大白话解释这个技术方案：我们的系统采用微服务架构...
/说人话 用生活化的方式向非技术人员解释什么是区块链
```

## 支持的场景

| 场景 | 说明 |
|------|------|
| 学术论文 | 解释创新点、技术贡献，让外行也能理解 |
| 技术方案 | 向非技术人员解释技术架构和选型 |
| 跨行业沟通 | 消除行业术语壁垒，建立共同理解 |
| 演讲准备 | 把专业内容转化为通俗易懂的演讲稿 |
| 日常工作 | 邮件、文档、会议沟通的通俗化 |
